# ModelConverter

Assimp を使用して、3Dモデル（FBX 等）を独自のゲームエンジン（DirectX 12）で高速に読み込めるカスタムバイナリ形式（`.bin` および `.anm`）に変換するコマンドラインツールです。

メッシュとアニメーションが一体になったファイルだけでなく、**アニメーションデータのみが含まれる FBX ファイル（メッシュなし）** からも自動的にモーションを抽出し、`.anm` ファイルとして独立して出力する機能を備えています。

## 機能特徴
- **フォーマットの分離**: 形状・マテリアル・骨組み情報を `.bin`、モーションデータを `.anm` に分けて出力。
- **アニメーション専用FBXのサポート**: メッシュが含まれない不完全なシーン（`AI_SCENE_FLAGS_INCOMPLETE`）であっても、アニメーションが存在すればパースを継続。
- **2種類のアニメーション出力**:
  - **ベイク方式（定数時間引き）**: 指定した FPS でフレームを等間隔サンプリング。ランタイム側は二分探索が不要になり高速にアクセス可能。
  - **スパース方式（キーフレーム保存）**: 元ファイルが持つキーフレームの時間と値をそのまま保存。容量を節約可能。
- **DirectX 12 向け最適化**: 頂点ウェイト数の制限（最大4ボーン）、インデックスの32bit/16bit自動切り替え、左手系への自動変換、UV反転などをサポート。
